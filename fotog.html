Fotogrametría

La fotogrametría es una técnica que permite obtener información precisa de objetos y terrenos a partir de fotografías.

Inicio

1. ¿Qué es?

Ciencia y técnica para obtener medidas y modelos 3D a partir de imágenes fotográficas.

2. Tipos de Fotogrametría

  • Aérea: Tomadas desde drones, aviones o satélites.
  • Terrestre: Imágenes captadas desde el suelo.

3. Principios Básicos

  • Uso de estereoscopía para generar profundidad.
  • Triangulación fotogramétrica.

4. Productos

  • Ortofotos
  • Modelos Digitales del Terreno (MDT)
  • Nubes de puntos 3D

5. Aplicaciones

  • Cartografía
  • Monitoreo ambiental
  • Obras civiles
  • Arqueología

6. Ventajas

  • Alta precisión
  • Cobertura rápida
  • Menor costo

7. Herramientas

  • Drones con cámaras RGB/multiespectrales
  • Pix4D, Agisoft Metashape, DroneDeploy
